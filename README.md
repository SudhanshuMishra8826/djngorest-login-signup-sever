# Instructions to run
1) Install all required tools
    pip install pipenv
    pipenv install
    pipenv shell
    pipenv install django
    pipenv install djangorestframework
    pipenv install djangorestframework-jwt
    pipenv install django-cors-headers

2) python manage.py runserver
